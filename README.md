# Algebra: Chapter 0

These are a collection of [Lean4](https://lean-lang.org/) proofs that will aim
to follow my solutions to [Aluffi](https://www.math.fsu.edu/~aluffi/)'s
[Algebra: Chapter 0](https://bookstore.ams.org/gsm-104). It is an excellent book
on top of also being very categorically minded, so if you choose to use it as a
first introduction to algebra, you will get very good exposure to all the
abstract nonsense you will need for a lifetime of mathematics.
